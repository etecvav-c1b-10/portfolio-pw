Questão 4 (Feito por Fernando Augusto):
Os padrões da Web permitem que sistemas diferentes consigam se comunicar na internet.

Explique:

Por que padrões como HTTP, HTML e DNS são importantes.

Os padrões DNS, HTTP e HTML são a base da internet porque criam uma linguagem universal: o DNS traduz nomes de sites em números (IP), o HTTP define as regras para enviar esses dados e o HTML organiza como tudo aparece na sua tela, garantindo que a rede seja compatível, rápida e acessível em qualquer dispositivo do mundo

Como esses padrões ajudam a garantir que diferentes navegadores e servidores funcionem corretamente.

Os padrões garantem a interoperabilidade ao funcionar como um manual de instruções universal: como navegadores (Chrome, Safari) e servidores seguem as mesmas regras de HTTP para enviar dados e HTML para exibir conteúdo, eles conseguem "falar a mesma língua", assegurando que qualquer site funcione e apareça corretamente em qualquer dispositivo ou software do mundo.
